一、【生理】以「榮衛—氣血—表裏—臟腑」為辨證框架#

《桂林古本》一開篇即把臨床判斷落在氣血盛衰與臟腑偏勝：氣血盛則脈強滑，衰則脈微濇；並強調「欲知病源，當憑脈變；欲知病變，先揣其本」。這裡的「本」可理解為：

1. 榮衛與表裏

- 榮在脈中、衛在脈外；外感之邪先犯肌表，故表證常見惡寒、發熱、身痛、無汗或自汗等；

- 「發熱惡寒」多屬陽（表/衛），「無熱惡寒」可偏陰（裏/臟）。

2. 氣血與津液

- 發汗、吐、下皆會影響津液。書中反覆警告：汗多可「亡陽」、下多可「傷陰」，而津液的存亡會改變證候（如煩躁、口渴、小便不利、筋攣等）。

3. 六經作為病位/病勢的時間軸

- 傷寒例中以脈象配合六經病位：太陽（尺寸俱浮）、陽明（尺寸俱長）、少陽（尺寸俱弦）、太陰（尺寸俱沉濡）、少陰（尺寸俱沉細）、厥陰（尺寸俱弦微），並以「逐日淺深」觀其傳變。

- 其精神是：同一類症狀（如發熱）必須落在病位與正邪盛衰上，才能決定治法。

4. 治法不是「對症」，而是「順勢」

- 書中有總綱：「表未解不可下；表已解而大滿大實、腹堅燥屎，乃可下。」又云「陽盛陰虛，汗之則死，下之則愈；陽虛陰盛，汗之則愈，下之則死」——這其實是把治法建立在寒熱/虛實的生理基礎上。

小結：在生理層面，《傷寒雜病論》的辨證不是抽象分類，而是以榮衛表裏、氣血津液、臟腑功能為底，透過脈象與症狀聯動去推斷「邪在何處、正氣如何、津液是否可承受」，再決定汗吐下與方藥。

二、【病理】從「邪入層次」與「誤治變證」建立病機推演#

1) 外感傳變：由表入裏、由經入腑#

《傷寒例》提供極具操作性的「症狀—脈—選方」對應，能直接拿來做臨床推理的骨架：

  • 太陽受邪（表實或表虛）：發熱惡寒、頭項強痛、身疼；可見無汗（多偏傷寒表實）或自汗（多偏中風表虛）。
  • 陽明：身熱汗出、口渴舌燥；進一步可走向「胃家實」而便秘、腹滿、潮熱、讝語等。
  • 少陽：往來寒熱、胸脇苦滿、口苦咽乾、目眩；屬半表半裏樞機不利。
  • 三陰（太陰/少陰/厥陰）：多見沉、細、濡、微等脈象，常伴下利、腹痛、厥冷、煩躁嘔逆等，提示正氣不足、寒熱錯雜或入裏更深。

這套病理並非要求「背六經條文」，而是要求：看到症狀先判病位（表/裏/半表半裏）與病性（寒/熱、虛/實），再判病勢（是否傳變、是否壞病）。

2) 壞病與誤治：汗吐下失度會「改寫證候」#

桂林古本大量篇幅談「可汗不可汗、可下不可下」的逆證，核心在於：

  • 誤汗：汗出過多 → 亡陽/傷津 → 惡寒、心悸、煩躁不得眠、筋惕肉瞤、甚至厥逆。
  • 誤下：表邪未解先下 → 邪陷入裏 → 結胸、痞、協熱下利、心下鞕滿等。
  • 吐下後的虛煩：出現心中懊憹、不得眠、反覆顛倒等，提示津液/胃氣受損，治法需轉向和解、清熱除煩或溫中復陽。

病理思路因此變成「動態」：

1. 原本是太陽表證，但被誤下 → 可能變成結胸/痞；

2. 原本是少陽樞機不利，但被誤下 → 柴胡證仍在且更複雜；

3. 原本是實熱可下，但若已亡津液 → 可能改走養陰/救津。

換句話說，《傷寒雜病論》的病理關鍵不是只有「邪正」，還有治療操作本身會造成的病機偏移。學開方必須把「既往用過什麼法（汗/吐/下/溫針等）」列入辨證要件。

三、【方藥】把「症狀→辨證→選方」落成可執行流程（含代表方）#

以下以《桂林古本》的條文精神整理成「開方流程圖（文字版）」，並附代表方作為學習路標。重點放在選方邏輯，不落入穴位。

A. 第一步：先做「三分法」定位——表 / 半表半裏 / 裏#

1. 偏表（太陽層次）：惡寒明顯、身痛、頭項強、脈浮；

2. 半表半裏（少陽樞機）：往來寒熱、胸脇滿、口苦咽乾、眩；

3. 偏裏（陽明或三陰入裏）：但熱不寒、汗出渴、腹滿便秘；或下利腹痛、四肢厥冷等。

原文有明確指引：「表未解不可下；表已解而大滿大實…乃可下。」這是選方之前的安全閥。

B. 第二步：再用「寒熱 + 虛實 + 津液」決定治法（汗/和/清/下/溫）#

  • 表實（無汗、喘、身疼、脈浮緊）→ 發汗解表、宣肺平喘

- 代表：麻黃湯（發熱無汗煩躁等）

- 病機：寒束肌表、肺氣不宣。

  • 表虛（自汗、惡風、鼻鳴乾嘔、脈浮弱）→ 解肌和營衛

- 代表：桂枝湯（「嗇嗇惡寒、淅淅惡風、翕翕發熱…桂枝湯主之」）

- 病機：衛強榮弱、腠理不固，邪在肌表而正氣亦虛。

  • 少陽（往來寒熱、胸脇苦滿、口苦咽乾、目眩）→ 和解樞機

- 代表：小柴胡湯；若兼裡實可見大便不通、痞鞕等，可走向大柴胡湯。

  • 陽明熱盛（大熱汗出、口渴舌燥、脈洪大）→ 清氣分熱

- 代表：白虎湯；若熱結成實、腑氣不通（便秘腹滿、潮熱讝語）→ 承氣湯類（大/小/調胃承氣）。

  • 太陰（腹滿、下利、脈沉濡等，或濕與脾搏）→ 溫中健脾/化濕

- 代表：書中濕內泄瀉「泄瀉宜理中湯」；中滿可見白朮茯苓厚朴湯等。

  • 少陰/厥陰深層（厥冷、咽痛、煩躁嘔逆、寒熱錯雜等）→ 依脈證或溫回或清熱解鬱

- 例：救逆可見四逆湯/通脈四逆湯系統；厥陰嘔逆吐蚘可見烏梅丸。

C. 第三步：用「關鍵鑑別點」把相似症狀分開（實戰最重要）#

下面列三組最常混淆、但最能體現「症狀→辨證→選方」的鑑別。

#### 鑑別 1：同樣發熱惡寒——桂枝湯 vs 麻黃湯

  • 桂枝湯證（太陽中風/表虛）：

- 核心：自汗 + 惡風 + 脈浮弱（陰弱陽浮）+ 鼻鳴乾嘔

- 治法：解肌、調和營衛（發汗中寓斂汗）

  • 麻黃湯證（太陽傷寒/表實）：

- 核心：無汗 + 身疼痛 + 脈浮緊（寒束表）+ 可見喘

- 治法：發汗解表、宣肺平喘

→ 同是「發熱惡寒」，關鍵問兩件事：有汗無汗？惡風為主還是惡寒身痛為主？再合脈象。

#### 鑑別 2：同樣口渴煩躁——白虎湯（氣分熱） vs 承氣湯（腑實）

  • 白虎湯路徑：大熱、汗出、口渴、脈洪大，偏「熱盛傷津」；
  • 承氣湯路徑：熱結在裏、腹滿堅、便秘或潮熱讝語，偏「實結不通」。

→ 書中也提示「傳陽明…宜白虎湯。不差，與承氣湯」：先辨是否已結成「燥屎/大實」，再決定清熱或攻下。

#### 鑑別 3：下利腹痛——太陰中焦虛寒 vs 協熱下利/誤治變證

  • 若下利伴腹痛、喜溫、脈沉遲/濡弱、食不化 → 多偏中焦虛寒（理中湯類思路）；
  • 若先有外感表證未解而被誤下，出現下利不止、心下痞鞕、煩躁等 → 需回到「是否誤治」的病史辨證，可能用瀉心湯類、桂枝人參湯、救逆法等。

→ 《傷寒雜病論》最強的地方，是把「病史（曾汗/吐/下）」納入辨證，這點直接決定選方方向。

D. 第四步：把選方落在「主方 + 變方」而非堆疊藥味#

桂林古本常見做法是：

  • 先定主方（如小柴胡湯、桂枝湯、麻黃湯、白虎湯、承氣湯）；
  • 再按兼證加減（如小柴胡湯條下列「若渴…去半夏加人參…若咳…加五味子去生薑加乾薑…」等）。

學開方的重點因此變成：

1) 先把病位病性抓對（主方）；

2) 再辨兼證（加減）；

3) 同時避開禁忌（如「咽中乾燥者不可發汗」「表未解不可下」等）。

四、可直接拿來練習的「從症狀到處方」小抄（教學用）#

目的：讓初學者在面對外感/雜病相似表現時，有一套不靠背方歌、而能推理出方義的路徑。

1. 先問病勢與既往治法：第幾日？有無已汗/已下/已吐？（避免把壞病當正病）

2. 看三大主訴：

- 寒熱（惡寒/惡風/往來寒熱/但熱不寒）

- 汗（無汗/自汗/汗後變化）

- 便溺與口渴（小便不利/便秘/下利/渴不渴）

3. 配脈象做表裏寒熱虛實：浮/沉、緊/緩、數/遲、洪/微、弦/濡等

4. 鎖定病位後選主方：

- 太陽表：桂枝湯 vs 麻黃湯（依汗與脈）

- 少陽：小柴胡湯（往來寒熱、胸脇滿、口苦咽乾）

- 陽明：白虎湯（熱盛傷津）或承氣湯（腑實）

- 太陰：理中湯（泄瀉/中焦虛寒）等

- 少陰厥陰：四逆/通脈四逆、烏梅丸等（依厥逆/嘔逆/吐蚘、寒熱錯雜）

5. 最後才談加減與調護：覆取微汗、粥助藥力、禁忌飲食等。

---

五、結語：此項研究的學習成效指標#

  • 能用「表—半表半裏—裏」先定位，再以「寒熱虛實津液」決定治法；
  • 能以太陽（桂枝/麻黃）、少陽（柴胡）、陽明（白虎/承氣）三大樞紐，推演多數外感與其變證；
  • 能把「誤汗/誤下/誤吐」造成的壞病變證納入辨證，避免用方方向性錯誤；
  • 能理解仲景方並非藥味堆疊，而是「主方確立→兼證加減→禁忌守護」的臨床工程。
