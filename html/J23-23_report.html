研究摘要

本報告系統性地梳理了經方派（以倪海廈老師為代表）在臨床診療中的核心方法論。不同於時方派或溫病派，經方辨證以《傷寒論》與《金匱要略》為指導，透過嚴密的徵兆收集（十問）與層次劃分（六經），建立一套高度邏輯化的診斷與遣方流程。本研究旨在提煉從「觀察徵兆」到「確定方劑」的標準路徑。

生理框架：健康人的六大標準

在經方的生理觀中，判斷一個人是否健康（即生理機能正常），倪海廈老師提出了六大具體指標。這些指標反映了人體內陽氣與陰津的動態平衡：

1. 睡眠：能一覺到天亮。這代表心火能下行進入腎水（心腎相交），且肝藏血的功能正常。

2. 胃口：有正常的飢餓感，但不暴飲暴食。這代表中焦脾胃運化功能健全。

3. 口渴：飲水適量，且飲水能解渴。這代表身體津液分佈正常，氣化功能良好。

4. 體溫與手腳：手腳常年溫暖，且額頭冷、手腳熱。這代表陽氣能順利到達四肢末梢，無寒凝血瘀。

5. 排泄（大便）：每日定時排便，成形且順暢。代表大腸傳導功能正常，體內無積滯。

6. 排泄（小便）：顏色淡黃且量足。代表膀胱氣化功能正常。

病理機制：六經辨證與陰陽消長

當上述生理指標出現偏差時，即進入病理狀態。經方派使用「六經辨證」來界定病邪的位置與人體正氣的盛衰：

1. 三陽病（太陽、陽明、少少陽）：

- 太陽病：病在表，病理機制為衛氣被束。

- 陽明病：病在裡且實熱，病理機制為胃家實，燥氣傷津。

- 少陽病：病在半表半裡，病理機制為膽氣不升，三焦水道不暢。

2. 三陰病（太陰、少陰、厥陰）：

- 太陰病：脾胃虛寒，病理機制為濕阻與中陽不足。

- 少陰病：心腎陽虛，病理機制為「脈微細，但欲寐」，代表生命動力的嚴重衰竭。

- 厥陰病：上熱下寒，陰陽錯雜。

透過「十問」收集的徵兆（如：汗、熱、惡寒、渴、痛、大小便性狀等），醫師可以精準定位病機處於哪一經，並判斷是寒熱虛實的哪種組合。

方藥方向：對證下藥與精準配伍

經方的遣方思路遵循「觀其脈證，知犯何逆，隨證治之」：

1. 收集徵兆（收集證據）：利用倪氏十問（睡眠、胃口、口渴、汗、二便、體溫、婦女經期等）進行地毯式搜索。

2. 六經定性（建立邏輯）：將收集到的症狀套入六經框架。例如，若有發熱、汗出、惡風，定位為太陽中風。

3. 方證比對（選取武器）：對比《傷寒論》、《金匱要略》或《類聚方》。如「太陽中風」對應「桂枝湯」；「陽明熱盛」對應「白虎湯」。

4. 精準用藥：經方用藥精簡（通常 5-9 味），強調「藥對」的使用。例如：

- 桂枝與白芍：調和營衛。

- 麻黃與杏仁：宣降肺氣。

- 附子與乾薑：回陽救逆。

5. 療效評估：治療後再次檢查「六大健康指標」。若睡眠改善、手腳變暖，代表藥物方向正確（轉陰為陽）。

結論與生活建議

「經方要怎麼看病」的核心在於將複雜的疾病簡化為陰陽與六經的動態平衡。對於女主人 Catherine 而言，日常可以透過「六大健康標準」進行自我監測。若發現手腳冰冷或睡眠不佳，應意識到這是陽氣不敷使用的早期信號。在治未病方面，保持中焦溫暖（少飲冰水）與情緒平和（少陽氣暢），是維持經方生理平衡的最佳策略。
