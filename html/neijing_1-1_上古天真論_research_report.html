一、篇章概要

《上古天真論》是《黃帝內經·素問》第一篇，奠定全書理論基礎。核心議題： 1. 為何上古之人能活百歲而動作不衰？ 2. 為何今時之人半百而衰？ 3. 人體生理衰老的時間規律（七八定律） 4. 養生的層次與境界（真人→至人→聖人→賢人）

二、生理機制解析

2.1 腎氣：生命之本

核心論述：

「腎者主水，受五藏六府之精而藏之，故五藏盛，乃能寫。」

生理意義： - 腎是精氣總倉庫：五臟六腑的精氣最終都匯聚於腎 - 腎主生殖：腎氣充足 → 天癸至 → 生殖能力成熟 - 腎主骨生髓：腎氣衰 → 骨質疏鬆、牙齒脫落、行步不正

臨牀啟示： - 治療不孕、陽痿、早衰，必從腎論治 - 補腎不等於壯陽，而是「填精補髓」 - 腎虛的表現：齒槁、髮墮、筋不能動、形體重

2.2 天癸：生殖之精

定義：腎氣充盛到一定程度產生的生殖物質

女性七七律： - 7歲：腎氣盛 → 換牙、長髮 - 14歲 (二七)：天癸至 → 月經初潮 → 具備生育能力 - 21-35歲：生殖高峯期 - 35歲 (五七)：陽明脈衰 → 面容開始衰老 - 49歲 (七七)：任脈虛、太衝脈衰 → 絕經 → 無法生育

男性八八律： - 8歲：腎氣實 → 換牙 - 16歲 (二八)：天癸至 → 精氣溢寫 → 具備生育能力 - 24-40歲：生殖高峯期 - 40歲 (五八)：腎氣衰 → 髮墮齒槁 - 64歲 (八八)：天癸竭 → 齒髮去

臨牀應用： - 月經不調、閉經 → 調任脈、太衝脈 - 男性不育 → 補腎填精，促進「精氣溢寫」 - 延緩衰老 → 保護腎氣，延緩天癸衰竭

2.3 經脈系統

任脈： - 位置：前正中線 - 功能：「任主胞胎」，掌管女性生殖 - 病理：任脈虛 → 月經不調、不孕

太衝脈 (衝脈)： - 別名：「十二經脈之海」、「血海」 - 功能：調節全身氣血 - 病理：衝脈衰 → 月經減少、絕經

陽明脈 (胃經)： - 路線：經過面部 - 病理：陽明脈衰 → 面容憔悴、面焦

三、病理機制剖析

3.1 衰老的根本原因

上古之人為何長壽？

「法於陰陽，和於術數，食飲有節，起居有常，不妄作勞，故能形與神俱，而盡終其天年。」

今時之人為何早衰？

「以酒為漿，以妄為常，醉以入房，以欲竭其精，以耗散其真，不知持滿，不時御神，務快其心，逆於生樂，起居無節。」

五大致病因素： 1. 飲食無節：酗酒傷脾胃，耗氣血 2. 房事過度：「醉以入房，以欲竭其精」→ 腎精虧損 3. 起居無常：晝夜顛倒，違反生理節律 4. 情志失調：「務快其心」→ 慾望無節制，耗散真氣 5. 妄作勞：過度勞累，透支體能

3.2 精氣耗散的連鎖反應

路徑：

過度房事/情慾 → 腎精虧損 → 天癸提前衰竭 → 任脈虛、衝脈衰 → 月經紊亂/絕經
                                              ↓
                                        五臟六腑失養 → 面焦、髮白、筋弱

臨牀表現： - 面容：面焦（憔悴）、顏色暗沉 - 毛髮：髮墮（脫髮）、髮白（早白） - 筋骨：筋不能動、行步不正、腰膝酸軟 - 生殖：陽痿、早洩、月經不調、不孕

3.3 「三陽脈衰於上」的病理

三陽脈：手三陽 (大腸、三焦、小腸) + 足三陽 (胃、膽、膀胱)

衰退表現： - 六七 (42歲)：三陽脈衰於上 → 面皆焦、髮始白

為何衰於上？ - 陽氣本應上升濡養頭面 - 三陽脈衰 → 陽氣不足 → 頭面失養 → 早衰

臨牀意義： - 治療早白髮、面色晦暗 → 補陽氣、調三陽 - 使用方劑：補中益氣湯、六味地黃丸 + 補陽還五湯

四、養生層次論

4.1 四種養生境界

真人 (最高境界)： - 修煉方式：「提挈天地，把握陰陽，呼吸精氣，獨立守神」 - 成就：「壽敝天地，無有終時」 - 本質：超越生死，與天地同壽

至人： - 修煉方式：「淳德全道，和於陰陽，調於四時，去世離俗，積精全神」 - 成就：「遊行天地之間，視聽八達之外」 - 本質：延年益壽，超凡入聖

聖人： - 修煉方式：「處天地之和，從八風之理，適嗜慾於世俗之間，無恚嗔之心」 - 成就：「形體不敝，精神不散，亦可以百數」 - 本質：入世修行，活百歲

賢人： - 修煉方式：「法則天地，象似日月，辨列星辰，逆從陰陽，分別四時」 - 成就：「益壽而有極時」 - 本質：順應自然，延壽有限

4.2 養生八字訣

「恬淡虛無，真氣從之；精神內守，病安從來。」

恬淡虛無： - 心境平和，不被慾望牽引 - 情志調和，不患得患失

真氣從之： - 心靜則氣順，氣順則血和 - 真氣充足，百病不生

精神內守： - 精神不外散，專注當下 - 神守則精固，精固則氣充
