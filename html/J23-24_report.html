J23-24 研究報告｜問診十法（把「症狀」問清楚）

一句話結論

「問診十法」的價值不在於背十個問題，而在於用固定順序把病人的主訴轉成可辨證、可追問、可落地到方證/方藥的結構化資料，避免「症狀一堆、資訊卻不成系統」。

核心觀念（把十問當成「資料結構」而非聊天）

本次來源（倪海廈體系整理文與多篇轉載）呈現一致的十問骨架：睡眠、胃口、大便、小便、口渴、寒熱、汗、體力精神、性功能、女子月經。其本質是把全身狀態拆成幾個最能反映「陰陽/表裡/寒熱/虛實」與「津液、氣血、腑氣、營衛」的窗口。

  • 先問「睡眠」：看神、看心腎交、看夜間節律（固定時間醒常帶方向性）。
  • 再問「胃口」：看胃氣/中焦運化、嗜味偏好（酸苦甘辛鹹）與欲食/厭食。
  • 便尿與口渴：看津液代謝、熱/寒、腑氣通降與水道。
  • 寒熱與汗：看表裡、營衛、陰陽盛衰與是否有「開闔失司」。
  • 體力精神：看氣機、正氣強弱、病程消耗。
  • 性功能/月經：看腎精、衝任、肝血與長期體質（慢病常在此露餡）。

來源摘意：十問的重點是「請按這個方式回答」，否則會變成雞同鴨講；中醫開方依症狀而非西醫檢驗報告（多篇轉載內容一致）。

生理（框架）：十問各自對應的身體運作

以下用「生理」角度，把每一問對應的系統與要抓的變數寫成可操作表單。

1) 睡眠（神志/陰陽出入） - 目的：看心神、肝魂、腎精與陰陽交替是否順。 - 主要欄位：入睡難/易醒/多夢/早醒；是否固定在某時辰醒；醒後口乾、心悸、盜汗。 - 追問：醒來是清醒還是昏沉？醒後能再睡嗎？

2) 胃口（胃氣/中焦運化） - 目的：看胃氣存亡、運化強弱、偏嗜與寒熱。 - 欄位：饑不欲食/食欲旺；喜冷/喜熱；喜辛辣/甜/鹹等；飯後腹脹/噯氣/反酸。

3) 大便（腑氣通降/津液入腸） - 目的：看腸道傳導、津液是否足、是否有熱結或寒凝。 - 欄位：頻次、形態（乾結/黏滯/稀溏/下利）、顏色、臭味、是否不爽、是否便後乏力。

4) 小便（水道/膀胱氣化） - 目的：看腎與膀胱氣化、津液代謝與熱/寒。 - 欄位：色（清長/黃赤）、量、夜尿、頻急、排尿無力、尿不出或無尿意。

5) 口渴（津液與熱象、飲食偏好） - 目的：看是否真渴/假渴、是否「渴欲飲」以及喜冷喜熱。 - 欄位：渴不渴；渴則飲量；最想喝冷/溫/熱；飲後能否止渴。

6) 寒熱（陰陽偏性/表裡） - 目的：抓寒象/熱象、四肢末梢循環與表證。 - 欄位：怕冷/怕熱；手足冷/手足心熱；是否午後潮熱；是否惡風惡寒。

7) 汗（營衛與開闔） - 目的：看表虛/表實、衛氣固攝、陰虛內熱。 - 欄位：自汗/盜汗；運動汗、緊張汗；汗出部位；汗後是否怕風、是否口渴加重。

8) 體力精神（正氣與病程消耗） - 目的：看氣虛/陽虛/陰虛或痰濕困阻。 - 欄位：晨起精神；午後倦；專注力；動則氣短；是否懶言。

9) 性功能（腎精與氣血） - 目的：抓腎氣腎陽腎陰、肝鬱、血瘀等長期體質訊號。 - 欄位：性慾、勃起/射精、帶下、疼痛、疲乏與情緒壓力。

10) 女子月經（衝任/肝腎/氣血） - 目的：看衝任是否調、血海盛衰與寒熱虛實。 - 欄位：週期（提前/延後）、量、色、血塊、痛經、經前乳脹、經後乏力。

病理（機制）：十問怎麼導向辨證

把十問收集到的資料，轉成「病理」推論時，建議遵循三步：

A) 先定大方向：表裡/寒熱/虛實/陰陽 - 寒熱：喜熱飲且不渴、畏寒、便溏 → 偏寒；渴喜冷飲、尿黃便秘 → 偏熱。 - 虛實：精神差、動則乏力、自汗 → 偏虛；便秘腹滿、身熱不汗、煩躁拒按 → 偏實。 - 表裡：惡寒發熱、汗出與否、身痛 → 表；腹滿便秘、口渴尿少 → 裡。

B) 再抓關鍵病機： - 津液：口渴＋尿少黃＋大便乾 → 津傷或熱盛；不渴＋尿清長＋便溏 → 陽虛或寒濕。 - 氣機：胸脅不舒、嗳氣、情志波動、便秘與腹脹交替 → 肝鬱氣滯。 - 痰濕：困倦、頭重、納呆、便黏不爽 → 痰濕中阻。 - 血瘀：痛有定處、經血暗有塊、舌質紫暗（此為延伸四診）→ 血瘀。

C) 連到方證（以《傷寒論》/經方思路做「下一步」） 十問本身不等於六經辨證，但它提供六經辨證的必要資料： - 寒熱、汗、口渴、便尿 → 直接對應「表/裡」「津液」「熱結/寒化」等關鍵。 - 睡眠、體力精神 → 反映正邪消長、虛實與病程。

舉例（示意，不替代醫療診斷）： - 若見惡寒、身痛、無汗、口不渴、尿清、精神尚可：可先把資訊整理成「表實偏寒」的路徑，後續再配合脈象/舌象判定是否類似麻黃湯證。 - 若見大便秘結、口渴、尿黃、身熱、汗出或不出、煩躁：可整理成「裡實熱結/津傷」路徑，後續評估是否有承氣湯類（大承氣/小承氣/調胃承氣）方向。 - 若見自汗、畏風、疲倦、胃口差：可整理成「表虛/氣虛」路徑，後續再看是否走桂枝湯類或補益調中方向。

方藥（方向）：把十問轉成「治法—方藥候選池」

這裡用「方向」而非硬套方名，因為十問缺少舌脈與體表細節；但它足以建立候選池，讓後續四診補齊即可落地。

1) 表證為主： - 惡寒發熱、身痛、汗出與否 → 桂枝湯系（表虛）、麻黃湯系（表實）作為候選方向。

2) 裡熱/熱結： - 便秘、口渴、尿黃、腹滿拒按 → 承氣湯系作為候選方向。

3) 中焦虛寒/寒濕： - 不渴或喜熱飲、便溏、尿清長、納差腹脹 → 理中/四逆或溫中化濕類方向作為候選。

4) 津傷陰虛： - 渴甚飲不解、便乾、盜汗、睡眠差 → 養陰生津/清熱或滋陰類方向（需再以舌脈核對）。

5) 肝鬱氣滯/血瘀（多見於月經/痛證線索）： - 經前乳脹、情緒波動、痛有定處、血塊 → 疏肝理氣、活血化瘀方向（再配合脈舌）。

可落地的問診表單（每問：目的→常見答案→辨證方向→下一步追問）

（以下可直接複製到臨床/自我紀錄用）

  1. 睡眠 - 目的：神志/陰陽 - 常見答案：難入睡｜易醒｜固定時辰醒｜多夢 - 辨證方向：陰虛火旺｜肝鬱｜心腎不交｜痰熱擾心 - 追問：醒後口乾？心悸？盜汗？

  2. 胃口 - 目的：胃氣/中焦 - 常見答案：饑不欲食｜嗜冷｜嗜熱｜飯後脹 - 辨證方向：中焦虛寒｜胃熱｜痰濕｜食積 - 追問：噯氣反酸？惡心？

  3. 大便 - 目的：腑氣/津液 - 常見答案：便秘乾結｜黏滯不爽｜稀溏下利 - 辨證方向：裡熱結｜津虧｜寒濕｜脾虛 - 追問：便後是否乏力？腹痛喜按或拒按？

  4. 小便 - 目的：氣化/水道 - 常見答案：尿黃少｜尿清長｜夜尿多｜頻急 - 辨證方向：熱｜寒/陽虛｜腎氣不固 - 追問：口渴與否？腰膝酸？

  5. 口渴 - 目的：津液/熱象 - 常見答案：渴喜冷｜渴喜熱｜不渴 - 辨證方向：熱盛津傷｜陽虛內寒｜痰濕壅遏 - 追問：飲後能否止渴？

  6. 寒熱 - 目的：陰陽/表裡 - 常見答案：畏寒｜怕熱｜手足冷｜手足心熱 - 辨證方向：寒證｜熱證｜陰虛｜陽虛 - 追問：是否惡風？是否午後加重？

  7. 汗 - 目的：營衛/開闔 - 常見答案：自汗｜盜汗｜不汗 - 辨證方向：表虛｜陰虛｜表實 - 追問：汗後怕風？口渴加重？

  8. 體力精神 - 目的：正氣盛衰 - 常見答案：晨起累｜午後倦｜動則氣短 - 辨證方向：氣虛｜陽虛｜痰濕困脾 - 追問：是否懶言？是否胸悶？

  9. 性功能 - 目的：腎精/氣血 - 常見答案：性慾低｜勃起弱｜疲乏 - 辨證方向：腎陽虛｜腎陰虛｜肝鬱 - 追問：腰膝酸？畏寒？口乾？

  10. 月經（女性） - 目的：衝任/血海 - 常見答案：提前｜延後｜血塊｜痛經 - 辨證方向：氣滯血瘀｜寒凝｜血虛｜血熱 - 追問：經前/經後哪個更不適？喜熱敷或拒按？
