# 中醫學習筆記（由淺入深）

這個 repo 用來整理女主人（Catherine）的中醫學習筆記。

- 索引依「系統性 × 學習難易度」分層：**入門 → 中階 → 高階 → 艱深**
- 每個項目都提供：**研究報告（完整）**＋**500 字摘要（先讀）**
- HTML 以 `html/` 目錄存放，建議用「HTML 預覽連結」直接閱讀（GitHub 直接點 `.html` 會顯示原始碼）。

## 如何閱讀
1. 先從 **500 字摘要**開始
2. 再看 **研究報告**（每份都用「生理／病理／方藥」三段架構）
